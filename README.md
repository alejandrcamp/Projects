# Análisis de Movilidad Urbana y PIB en Latinoamérica (2024)

## Objetivo
Identificar si existe una relación entre movilidad urbana y desempeño económico, así como detectar patrones y posibles oportunidades de mejora en infraestructura de transporte.

## Herramientas utilizadas
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
El análisis incluye información de movilidad urbana y variables económicas de 15 ciudades pertenecientes a 7 países de Latinoamérica: Argentina, Brasil, Chile, Colombia, México, Perú y Uruguay.

## Hallazgos principales
- Existe una relación positiva débil entre congestión vehicular y PIB per cápita.
- Montevideo destacó por combinar baja congestión y alto PIB.
- Santiago presentó congestión moderada y el PIB más bajo del análisis.
- Los resultados sugieren que la productividad económica depende de múltiples factores además del tráfico.
